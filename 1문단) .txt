1문단) 

In 특허법 : 발명을 한자 또는 승계인/독점적 배타적 권리 = 특허권 -> 대가를 지불하고 기술 공개 

when? 특허출원날부터 1년 6개월 되는날 

what? 대중에게 공개되어/누구든지/특허정보검색 서비스에 접속하여/특허 출원기술 내용 열람 가능 

원칙) 누구든지 -> 열람 기술 내용 개량하여 발명가능 -> 개량된 발명에 대한 특허권 취득 가능 

​

2문단) 

특허권 부여의 예외사항: 모든사람X 

조건 1) 이전 발명에 비해 새롭고 기술적 면에서 발전이 있는 발명을 대상 

         2) 특허신청, 즉 특허출원 후 특허등록 완료후 부여 

​

특허출원자 = 발명자 + 발명에 대한 권리 승계받은 사람 

특허출원의 대상 = 물건의 발명 + 영업방식등의 방법도 포함 

​

특허등록의 과정 

(1) 특허청이 특허출원에 대한 판정 -> 특허 등록 -> 

(2) 출원자-> 독점적 이용할 수 있는 권한 가짐 = 이요은 - 산업적 상업적 이용으로만 한정 ='실시'라 표현 

(3) 특허권자 - 실시 권한 양도 가능 -> 특정인에게만 양도 -독점적 실시권의 양도 or 여럿에게 양도 - 통상적 실시권의 양도 -> 사용료 (실시료) 받음

​

3문단) 

특허권 침해 = 특허권에 대한 정당한 권한이 없는 자가 타인의 특허권을 가진 발명을 이용하는 것 (실시) 

특허권 침해행위 = 민법상 재산권 침해행위 => 불법행위 => "고의 또는 과실로 인한 위법행위" -> 손해배상의 책임 (민법 제750조) 청구가능

입증책임 - 피해자에게 존재 

but, 특허권침해행위 = 특허권자가 입증하기 어려운 문제점(P) -> (S) 보호를 위해 입증책임 경감 규정 마련 = 생산방법의 추정규정 

'생산방법의 추정규정' = 물건을 생산하는 방법의 발명에 관해, 물건과 동일한 물건은 특허 등록된 방법에 의하여 생산된 것으로 추정 

​

4문단) 

(1)In 특허법) 손해액 산정과 관련 '일실이익추정규정' 

일실이익 = 특허권 침해가 없었다면 얻을 수 있었던 특허권자의 이익액, 즉 특허권자가 실제로 입은 손해액

if 특허권 침해가 없었을 때,더 판매할 수 있었던 물건의 수량 * 단위 수량당 이익액 (X) 

(특허권자가 실제 생산할 수 있던 물건의 수량 - 실제 판매한 물건의 수량 ) * 단위 수량당 이익액(O)

​

(2)침해자가 침해행위를 통해 얻은 이익액을 손해액으로 추정하는 규정 

특허권자- 입증 -> 이익액을 손해액으로 추정  -> 입증책임의 전환 -> 책임자 자신의 이익액이 영향이 없음을 입증 

​

(3) 통상 실시료(이용료)를 통해 손해액을 추정할 수 있도록 하는 규정 

특허권자가 실제로 입은 손해, 즉 일실이익과 관계없이 최소배상액 규정을 위한 통상 실시료 정함. 

통상실시료 : 판매액* 실시료율
